# Git Commit Command

Create a git commit for changes made in the current session.

## Instructions

1. First, run `git status` to show what files have changed
2. Run `git diff --stat` to show a summary of changes
3. Run `git diff` to show the detailed changes (truncate if too long)
4. Stage all modified and new files using `git add .`
5. Create a meaningful commit message based on the changes you observe
6. Execute the git commit with the generated message
7. Show the final git log entry to confirm the commit
